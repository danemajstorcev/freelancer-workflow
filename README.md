Freelancer Workflow


Project Description:

Freelancer Workflow is a productivity web application designed to help freelancers manage tasks, clients, and daily workflows in a structured and organized way. The project focuses on clean UI design and efficient state management.


Tech Stack / Built With:

- React
- TypeScript
- styled-components


Features:

- Task management system
- Workflow and productivity organization
- Clean and responsive UI
- Modular component-based structure


Getting Started / Installation:

- Clone the repository:
git clone https://github.com/danemajstorcev/freelancer-workflow.git

- Navigate to the project folder:
cd freelancer-workflow

- Install dependencies:
npm install

- Run the project:
npm run dev
